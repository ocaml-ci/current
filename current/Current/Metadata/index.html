Module Current.Metadata

Metadata associated with primitive terms.

type t = {
  1. job_id : job_id option;
    (*

    The job ID, for jobs with logs. This is used when generating the diagrams to create links to the job pages, and can also be useful for indexing.

    *)
  2. update : Current_term.Output.active option;
    (*

    If the job is updating in the background (while still outputting the previous value), this gives the status of the update. In the diagrams, this will appear as a gradient from the update colour on the left to the output state's colour on the right.

    *)
}
